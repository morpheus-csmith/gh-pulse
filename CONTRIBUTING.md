## Quick Start

1. Fork the repo
2. Clone your fork
3. Install dependencies: `npm install`
4. Make your changes
5. Test: `npm test`
6. Submit a PR

## Code Style

- Use meaningful variable names
- Add comments for complex logic
- Run tests before submitting
